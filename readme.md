## 2. 一些程序选用的定历规则

- 朔气同日，朔在前
- 纪年干支以春节为界，纪月干支以朔日为界（闰月同上月）
- 四柱：年柱以立春为界，月柱以节为界（建寅），日柱以零点为界，时柱以北京时间为准，未考虑地方时
